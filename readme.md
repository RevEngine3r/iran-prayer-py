# اوقات شرعی ایران (روش تهران)

کتابخانه‌ی پایتون (نسخه ۳.۱۰ به بالا) برای محاسبه‌ی اوقات شرعی در ایران بر اساس **روش تهران** (مؤسسه ژئوفیزیک دانشگاه تهران).  
بدون کتابخانه‌های خارجی، بدون کدهای C — فقط پایتون خالص.

## امکانات
- محاسبه‌ی دقیق **اذان صبح، طلوع خورشید، اذان ظهر، عصر، غروب خورشید، اذان مغرب، عشاء، نیمه شب شرعی**.
- استفاده از زوایای رسمی روش تهران (فجر ۱۷٫۷°، عشاء ۱۴°).
- پشتیبانی از شهرهای اصلی ایران با `Enum` (`IranCity`).
- تنظیم خودکار منطقه زمانی ایران (UTC+3:30).
- امکان تغییر فاصله‌ی مغرب (پیش‌فرض +۱۹ دقیقه مطابق تقویم‌های رسمی ایران).
- نوشته شده با پایتون ۳.۱۰+ و سینتکس مدرن (`match/case`، تایپ‌هینت‌ها، `dataclass`).

## بدون وابستگی خارجی
فقط از کتابخانه‌های استاندارد پایتون استفاده می‌شود.

## استفاده

```python
from iran_prayer_times import IranPrayerTimes, IranCity

# انتخاب شهر
ipt = IranPrayerTimes(IranCity.TABRIZ)

# محاسبه برای امروز
times = ipt.compute()

print("اذان صبح:", times.fajr.strftime("%H:%M"))
print("طلوع خورشید:", times.sunrise.strftime("%H:%M"))
print("اذان ظهر:", times.dhuhr.strftime("%H:%M"))
print("عصر:", times.asr.strftime("%H:%M"))
print("غروب خورشید:", times.sunset.strftime("%H:%M"))
print("اذان مغرب:", times.maghrib.strftime("%H:%M"))
print("عشاء:", times.isha.strftime("%H:%M"))
print("نیمه شب شرعی:", times.midnight.strftime("%H:%M"))
```

## نکات
- **اذان مغرب**: به طور پیش‌فرض ۱۹ دقیقه بعد از غروب خورشید (مطابق تقویم‌های رسمی ایران).
- **نیمه شب شرعی**: وسط زمان بین غروب خورشید و اذان صبح روز بعد.
- کاملاً آفلاین و بدون نیاز به اینترنت.
